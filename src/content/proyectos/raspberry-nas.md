# Raspberry Pi NAS seguro

Este proyecto consiste en construir un sistema de almacenamiento seguro utilizando una Raspberry Pi.

## Objetivo

Crear una infraestructura self-hosted accesible remotamente mediante VPN.

## Tecnologías utilizadas

- Raspberry Pi 5
- Nextcloud
- WireGuard
- Nginx

## Conclusiones de seguridad

El acceso a los servicios se limita exclusivamente mediante VPN, evitando exposición directa a internet.
